# Example gallery — publication-quality figures

A gallery of **20 modern, high-impact-journal-ready figures**, each generated from **synthetic (seeded, reproducible) data** by [`generate_gallery.py`](generate_gallery.py). Every figure demonstrates one style from the [graph-style library](../references/graph-style-library.md) and is built to the skill's own publication-ready rules:

- **One colorblind-safe palette** (Okabe-Ito) reused across every figure — orange = key finding, blue = comparator, grey = control.
- **Individual data points shown on bars** (bars never hide the sample), error-bar meaning stated, **exact P-values**.
- **Sequential, perceptually-uniform colormaps** (viridis) for fields — never jet/rainbow.
- **Sans-serif typography, clean de-spined axes**, vector output.
- **Domain-correct statistics**: geometric mean + log axis for PK, f2 for dissolution, parity + agreement for models.

Each figure is exported as **PNG (300 dpi)** for preview and **PDF (vector)** for submission — see [`png/`](png/) and [`pdf/`](pdf/). Regenerate everything with `python3 generate_gallery.py` (needs only matplotlib + numpy + scipy; no LaTeX).

> The data are synthetic and illustrative — the figures demonstrate **form and standards**, not real results.
